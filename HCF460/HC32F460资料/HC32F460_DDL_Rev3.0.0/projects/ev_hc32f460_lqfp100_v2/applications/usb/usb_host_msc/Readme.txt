平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示MCU USB外设实现的的USB host MSC功能。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手软件（115200bps，8N1）

================================================================================
使用步骤
================================================================================
1）打开工程编译并全速运行；（如果需要调试文件系统功能，请自行添加第三方文件系统源码，并使能USB_MSC_FAT_VALID宏定义）
2）J14连接OTG转接线，再接上U盘；
3）识别到U盘后,串口打印U盘信息；
4）如已添加文件系统源码及使能USB_MSC_FAT_VALID宏定义，则led闪烁，串口打印文件信息；
5）如已添加文件系统源码及使能USB_MSC_FAT_VALID宏定义，根据串口输出提示短按K10按键进行文件读写操作。

================================================================================
