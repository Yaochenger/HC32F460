平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示MCU的CDC VCP 和 HID的复合设备功能。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------

辅助软件:
---------------------
串口调试助手软件（115200bps，8N1），Bus Hound

================================================================================
使用步骤
================================================================================
1）使用USB连接线通过测试版的J14连接到PC，
2）打开工程编译并全速运行（可使用自带的虚拟串口观察调试信息）；
CDC功能
3）安装CDC VCP driver
4）第三步成功后，windows设备管理器会出现虚拟COM口
5）串口数据可用过EVB上的J2收发
HID功能
6）点击按键K10 HID设备会通过IN端点发出按键信息，
7）在PC端使用Bus Hound通过OUT端点向设备发送数据可以控制LED的状态
     发送 0x01 0x00 熄灭红色LED
     发送 0x01 0x01 点亮红色LED
     发送 0x02 0x00 熄灭蓝色LED
     发送 0x02 0x01 点亮蓝色LED
