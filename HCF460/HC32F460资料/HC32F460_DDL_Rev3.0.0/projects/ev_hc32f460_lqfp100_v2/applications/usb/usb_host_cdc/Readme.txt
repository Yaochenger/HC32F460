平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示MCU USB外设实现的USB host CDC功能，HOST将收到的设备CDC数据进行回发，
以验证通信是否成功。

说明：

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手软件（115200bps，8N1）

================================================================================
使用步骤
================================================================================
1）打开工程编译并全速运行在主机开发板上，可使用板载虚拟串口查看运行输出信息；
2）J14连接OTG转接线，再接上CDC设备；
  （使用另一块HC32开发板烧录usb_dev_cdc样例获得该CDC设备）
3）设备枚举完成后，主机开发板led闪烁；
4）CDC设备的串口数据收发端口，通过USB转串口工具接到PC串口助手；
5）串口助手以（115200bps，8N1）格式发送数据，可以收到相同数据回显。

================================================================================
注意
================================================================================
1）为了保证供电充足，注意将主机板子J16 PWRSEL的DAP跳线短接；
2）CDC设备的串口数据收发使用的端口，请参考对应的usb_dev_cdc样例说明。
