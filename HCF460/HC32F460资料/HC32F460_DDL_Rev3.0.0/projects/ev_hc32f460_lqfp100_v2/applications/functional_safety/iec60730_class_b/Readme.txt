平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例为IEC60730_STL_ClassB测试样例。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
杜邦线

辅助软件:
---------------------
串口调试助手软件（115200bps，8N1）

================================================================================
使用步骤
================================================================================
1）测试板3V3供电；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行
4）观察串口调试助手打印信息；
5）旋转电位器ADC：ADC输入电压[1.6V~1.7V]范围(即ADC采样值(1993,2102)窗口)外，打印信息提示ADC测试异常；
6）旋转电位器ADC：ADC输入电压[1.6V~1.7V]范围(即ADC采样值(1993,2102)窗口)内，ADC测试异常信息停止打印；
7）按下按键K10：打印信息提示GPIO输入测试异常；
8）释放按键K10：GPIO输入测试异常信息停止打印；
9）重复步骤4）~8）
