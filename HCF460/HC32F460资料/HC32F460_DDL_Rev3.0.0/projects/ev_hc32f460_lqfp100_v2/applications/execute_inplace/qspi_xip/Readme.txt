平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示将代码下载到QSPI外挂的Flash中执行的定时功能。

说明：
本样例设置SysTick工作频率为1000Hz，即每1毫秒触发一次计数溢出中断，使用SysTick作
精准延时控制，LED_RED和LED_BLUE交替闪烁，其中LED闪烁代码位于QSPI外挂flash中。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，关闭IDE下载界面；
3）观察LED_RED和LED_BLUE交替闪烁。

================================================================================
注意
================================================================================
1.使用MDK工程时，请将MDK路径下的“config\flashloader\HC32F460_qspi.FLM”文件复制到
KEIL对应的路径下(通常为Keil_v5\ARM\Flash)，并在Option->Utilities->Settings->
Flash Download->Add里添加“HC32F460 QSPI Flash”。
2.样例暂不支持GCC编译环境。
