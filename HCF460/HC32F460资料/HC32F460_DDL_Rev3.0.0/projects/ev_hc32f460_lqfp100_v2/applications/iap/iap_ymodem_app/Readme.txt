平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示基于YModem通信IAP的APP功能。

================================================================================
测试环境
================================================================================

---------------------

辅助工具:
---------------------
Micro USB数据线

辅助软件:
---------------------
HyperTerminal

================================================================================
使用步骤
================================================================================
1）先执行iap_ymodem_boot样例中readme.txt的操作；
2）打开工程，重新编译后关闭IDE，在工程路径下生成“bin文件”；
3）使用USB线连接电脑及目标板，在“HyperTerminal”软件上打开USB对应串口，配置端口如下参数：
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
       流控：None
4）按下目标板的复位按键，可看到打印出操作菜单，在HyperTerminal窗口中输入“1”，进
入程序下载流程；
5）点击软件菜单栏的“传送”->“发送文件”，协议选择“Ymodem”，然后文件名选择步骤2生
成的“bin文件”，然后点击“发送”，等待发送完成后会自动关闭发送窗口；
6）在HyperTerminal窗口中输入“3”，跳转到app程序；
7）跳转成功后自动运行APP程序，观察LED_RED和LED_BLUE交替闪烁。
