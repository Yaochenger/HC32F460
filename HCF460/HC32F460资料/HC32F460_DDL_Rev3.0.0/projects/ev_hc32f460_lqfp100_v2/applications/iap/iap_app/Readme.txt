平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示IAP的APP功能。

================================================================================
使用步骤
================================================================================
1）先执行iap_boot样例中readme.txt的操作；
2）打开工程，重新编译后关闭IDE，在工程路径下生成“bin文件”；
3）使用USB线连接电脑及目标板，在“IAP_Demo”软件上打开USB对应串口，配置端口如下参数：
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
4）参照用户手册修改“目标地址”，在“文件路径”处选择步骤2生成的“bin文件”；
5）点击“连接”按钮，连接成功后，再点击“下载”按钮，下载app程序；
6）下载成功后自动运行APP程序，观察LED_RED和LED_BLUE交替闪烁。
