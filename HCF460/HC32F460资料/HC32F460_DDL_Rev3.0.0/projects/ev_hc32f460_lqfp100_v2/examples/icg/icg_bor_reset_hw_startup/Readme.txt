平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示ICG模块中BOR复位功能。

说明：
本样例设置硬件启动BOR且电压阈值为2.3V，正常启动后，调整供电电压低于阈值进入复位
状态；再次调整供电电压高于阈值进入正常工作状态，LED_RED闪烁1次。

无

================================================================================
使用步骤
================================================================================
1）打开并重新编译工程，启动IDE的下载程序功能，关闭IDE下载界面；
2）取下跳线帽J20，将稳压电源调节至3.3V，并通过J20（靠近芯片的一端）为MCU供电；
3）目标板使用USB数据线供电，按下复位按键复位芯片；
4）将稳压电源电压调至2.2V，此时芯片进入复位状态；
5）再将稳压电压调至3.3V，芯片退出复位状态并进入正常工作状态，检测是欠压复位，
LED_RED闪烁1次。
