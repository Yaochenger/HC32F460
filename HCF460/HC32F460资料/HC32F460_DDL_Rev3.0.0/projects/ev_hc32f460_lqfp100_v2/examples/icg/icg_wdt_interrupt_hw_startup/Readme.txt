平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示ICG模块中WDT中断功能。

说明：
本样例设置硬件启动WDT，并设置约1.05秒触发一次WDT计数溢出中断，通过按键K10循环切
换工作模式（正常模式、睡眠模式），正常模式LED_RED闪烁，睡眠模式LED_BLUE闪烁。

================================================================================
使用步骤
================================================================================
1）打开并重新编译工程，启动IDE的下载和调试功能，关闭IDE下载界面；
2）按下复位按键复位芯片，进入正常模式LED_RED闪烁；
3）按下按键K10切换工作状态到睡眠模式，LED_BLUE闪烁；
4）再次按下按键K10切换工作状态到正常模式，LED_RED闪烁。
