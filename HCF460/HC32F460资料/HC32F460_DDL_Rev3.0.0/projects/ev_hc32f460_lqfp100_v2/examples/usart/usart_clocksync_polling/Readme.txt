平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示USART外设配置为时钟同步模式时通过轮询方式收发数据。

================================================================================
使用步骤
================================================================================
1）需两个测试板，按照如下方式连接。
   主机:USART2  从机:USART2
    J5_GND------J5_GND
    J5_RX ------J5_TX
    J5_TX ------J5_RX
    J5_CK ------J5_CK
2）打开工程usart_clocksync_polling。
3）按照如下修改宏定义，编译、下载程序至主机，全速运行。
   #define MASTER_DEVICE_ENABLE            (DDL_ON)
4）按照如下修改宏定义，编译、下载程序至从机，全速运行。
   #define MASTER_DEVICE_ENABLE            (DDL_OFF)
5）按从机按键K1。
6）按主机按键K1。
7）主机和从机蓝色LED点亮，表示数据收发一致。

================================================================================
注意
================================================================================
LED状态说明：
   a）蓝色LED灯亮，表示USART初始化成功并且数据收发一致。
   b）红色LED灯亮，表示USART初始化失败或者数据收发不一致。
