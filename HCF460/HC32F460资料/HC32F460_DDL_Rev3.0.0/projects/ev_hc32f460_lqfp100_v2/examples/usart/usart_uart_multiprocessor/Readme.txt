平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示USART外设配置为UART多处理器模式时收发数据。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
串口助手软件

================================================================================
使用步骤
================================================================================
1）需要两个测试板（主机和从机），按照如下方式连接。
   主机:USART2   从机:USART2
    J5_GND ------ J5_GND
    J5_RX  ------ J5_TX
    J5_TX  ------ J5_RX
2）通过USB数据线，连接PC和板子DAP。
3）打开工程usart_uart_multiple_processor。
4）按照如下修改宏定义，编译、下载程序至主机，全速运行。
   #define MASTER_DEVICE_ENABLE            (DDL_ON)
5）按照如下修改宏定义，编译、下载程序至从机，全速运行。
   #define MASTER_DEVICE_ENABLE            (DDL_OFF)
6）按主机按键K1。
7）主机和从机蓝色LED点亮，表示数据收发一致。

================================================================================
注意
================================================================================
LED状态说明：
   a）蓝色LED灯亮，表示USART初始化成功并且数据收发一致。
   b）红色LED灯亮，表示USART初始化失败或者数据收发不一致。
