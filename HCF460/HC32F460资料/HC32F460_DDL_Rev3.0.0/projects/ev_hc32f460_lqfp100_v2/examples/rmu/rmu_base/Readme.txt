平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
1）读取复位信息，并通过串口发送；
2）通过KEY10或RESET KEY分别触发SW Reset和Pin Reset。

测试环境

无

辅助软件:
---------------------
串口助手软件

================================================================================
使用步骤
================================================================================
1）打开工程，Micro USB连接PC与目标板，并编译下载；
2）打开串口调试助手，配置端口如下参数：
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
3）按下KEY10、RESET KEY或开合J20；
4）观测串口助手打印信息，查看本次复位情况。
