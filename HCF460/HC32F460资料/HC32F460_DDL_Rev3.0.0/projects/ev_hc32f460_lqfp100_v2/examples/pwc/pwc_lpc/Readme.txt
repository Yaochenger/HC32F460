平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为芯片分别进入sleep模式，stop模式，power down模式3时的功耗测量。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
电流表

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）断开J20，且在J20串联接入电流表。
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行。（）
4）按下WKUP（K10）按键，可观察到电流表上功耗变化，具体变化如下（WKUP下的数字表示按键次数）：

---WKUP-------模式----------功耗----
----0---------RUN---------约850uA---
----1--------SLEEP--------约500uA---
----2---------RUN---------约800uA---   (当前RUN模式功耗更低，是由于PWC_LDO_Cmd(PWC_LDO_HRC, DISABLE)。)
----3---------STOP--------约60uA----
----4---------RUN---------约850uA---
----5------ PowerDown-----约1.8uA---
