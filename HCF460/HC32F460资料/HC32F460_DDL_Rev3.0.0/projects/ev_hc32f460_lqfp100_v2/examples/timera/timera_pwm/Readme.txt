平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程用TimerA单元1的通道1和通道2实现PWM输出。例程实现了三种PWM输出：
1. 一路普通的PWM（PA9），频率为200KHz，占空比为30%；
2. 两路单边对齐的PWM（PA8和PA9），频率均为100KHz，占空比分别为30%和55%；
3. 两路双边对称的互补PWM（PA8和PA9）频率均为50KHz，占空比分别为50%和40%。

说明：
1. TimerA钟源是PCLK1；
2. 配置不同的PWM时，注意其引脚的功能选择是否正确，详见“引脚配置和引脚功能”章节的
   引脚功能表。

================================================================================
使用步骤
================================================================================
1）将示波器探头连接至 PA8 和 PA9；
2）打开工程，重新编译，启动调试或直接下载程序运行；
3）可看到示波器显示一路频率为 200KHz、占空比为 30% 的 PWM；
4）将宏 APP_FUNC 修改为 APP_FUNC_SINGLE_EDGE_ALIGNED_PWM，输出两路单边对齐单边对齐
   PWM。重新编译，启动调试或直接下载程序运行。
   现象：示波器显示两路单边对齐的 PWM，频率均为 100KHz，占空比分别为 30% 和 55%。
5）将宏 APP_FUNC 修改为 APP_FUNC_TOW_EDGE_SYMMETRIC_PWM，输出两路双边对称的互补的
   PWM。重新编译，启动调试或直接下载程序运行。
   现象：示波器显示两路双边对称的互补 PWM，频率均为 50KHz，占空比分别为 50% 和 40%。
