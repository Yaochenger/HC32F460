平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程用TimerA单元1的通道1实现1ms定时，并将定时时序输出至PA2。

说明：
1. 系统时钟为MRC，8MHz；
2. TimerA时钟为PCLK0，8MHz；
3. 如果需要使用TimerA的硬件触发器，来触发TimerA启动或停止，请将‘TMRA_USE_HW_TRIG’定义
   为非0；
   1）例程将事件“EVT_SRC_PORT_EIRQ12”设置为TimerA的硬件启动条件，即该事件产生后，TimerA
      开始计数。按下开发板上的K1可产生此事件。
   2）例程将引脚PE7的下降沿设置为TimerA的硬件停止条件，即PE7上产生下降沿时，TimerA停止
      计数。

================================================================================
使用步骤
================================================================================
1）将示波器探头连接至 PA2；
2）打开工程，重新编译，启动调试或直接下载程序运行；
3）可看到示波器显示一路频率为 500Hz 的方波；
4）将 PE7 与 PB1（连着按键 K10）短接；
5）将宏 TMRA_USE_HW_TRIG 定义为非 0，使能硬件触发功能。重新编译程序，启动调试或直接
   下载程序运行，按下并释放按键 K1，可看到示波器显示一路频率为 500Hz 的方波，按下按键
   K10，可看到方波停止输出。
