平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程实现了CRC16和CRC32的硬件计算和软件计算。对CRC硬件计算和软件计算的结果进行
对比，一致则蓝灯亮，否则红灯亮。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程，编译、下载并运行程序；
2）观察LED蓝灯常亮，硬件计算和软件计算的结果一致；

================================================================================
注意
================================================================================
LED状态说明：
   a）蓝色LED灯亮，硬件计算和软件计算的结果一致。
   b）红色LED灯亮，硬件计算和软件计算的结果不一致。
