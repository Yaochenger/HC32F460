平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例为flash引导交换。

================================================================================
使用步骤
================================================================================
1）打开本样例工程，编译下载，全速运行，可观察到蓝灯点亮（SWAP功能未开启）；
2）按下按键K1，使能引导交换功能,然后复位MCU；
3）再次编译下载，全速运行，可观察到红灯点亮（SWAP功能开启）；
4）按下K2(SWAP功能关闭)，然后复位MCU，可观察到蓝灯点亮。
5）重复步骤2）和步骤4），可观察到红灯蓝灯交替闪烁。
