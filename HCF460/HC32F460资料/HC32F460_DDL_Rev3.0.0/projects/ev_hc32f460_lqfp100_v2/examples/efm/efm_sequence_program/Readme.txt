平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例为flash 连续编程，全擦除样例。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）可观察到蓝灯亮。

================================================================================
注意
================================================================================
1.确保程序在RAM里运行
2.使用MDK工程时，请将drivers\cmsis\Device\HDSC\hc32f4xx\Source\ARM\flashloader\HC32F460_RAM.FLM
  文件复制到KEIL的对应路径下(通常为Keil_v5\ARM\Flash)
  如果更改工程默认的debug方式，还需在Option->Utilities->Settings->Flash Download->Add里添加
  HC32F460 RAM文件，且修改RAM for Algorithm 框里Start address。
