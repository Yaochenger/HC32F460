平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示MPU写保护功能。

说明：
本样例展示对RTC内存区间写保护，触发保护时内存产生管理故障；

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
Micro USB数据线

辅助软件:
---------------------
串口调试助手

================================================================================
使用步骤
================================================================================
1）使用USB线连接电脑及目标板的J8 USBDAP端口；
2）在串口调试助手上打开USB对应串口，配置端口如下参数：
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
3）打开工程并重新编译，启动IDE的下载程序功能，关闭IDE下载界面；
4）按下按键K2，触发获取RTC时间，获取成功后通过串口打印输出，LED_BLUE亮起；
5）按下按键K3，触发设置RTC时间，因写保护故设置失败，触发内存管理故障，LED_RED亮
起，LED_BLUE熄灭；
