平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示MPU写保护功能。

说明：
本样例设置DMA2单元对Region 2区域写保护，触发写保护时产生NMI中断；
使用LED_BLUE表示收发数据是否相同，相同则LED_BLUE亮起，反之LED_BLUE熄灭；
使用LED_YELLOW表示MPU开关状态，LED_YELLOW亮起表示MPU开启，LED_YELLOW熄灭表示MPU
关闭；LED_RED亮起则表示触发NMI中断。

使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，关闭IDE下载界面；
3）按下按键K2，使能MPU保护功能，此时LED_YELLOW亮起；
4）按下按键K3，触发数据传输，因开启MPU保护，故数据无法写入，触发异常NMI中断，同
时数据对比不同，故LED_RED亮起；
5）再次按下按键K2，关闭MPU保护功能，此时LED_YELLOW和LED_RED都熄灭；
6）按下按键K3，触发数据传输，传输完成后LED_BLUE亮起则数据正常；反之，则通信错误；
7）可以重复测试步骤3-6。
