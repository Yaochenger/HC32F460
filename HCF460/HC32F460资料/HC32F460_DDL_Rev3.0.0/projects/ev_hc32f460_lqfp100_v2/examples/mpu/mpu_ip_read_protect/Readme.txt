平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示MPU IP读写保护功能。

说明：
本样例使能RTC单元读写保护，触发保护时产生总线故障；
其中LED_BLUE表示线程模式的特权级别，LED_BLUE亮则表示当前运行在特权级，LED_BLUE熄
灭则表示当前运行在非特权级别；获取时间触发总线错误则LED_RED亮起，反之则熄灭。

Micro USB数据线

辅助软件:
---------------------
串口调试助手

================================================================================
使用步骤
================================================================================
1）使用USB线连接电脑及目标板的J8 USBDAP端口；
2）在串口调试助手上打开USB对应串口，配置端口如下参数：
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
3）打开工程并重新编译，启动IDE的下载程序功能，关闭IDE下载界面；
4）上电默认工作在特权模式，故LED_BLUE亮起；
5）按下按键K2，触发获取当前时间，获取成功后通过串口打印输出；
6）按下按键K10，切换当前线程模式到非特权模式，LED_BLUE熄灭；
7）按下按键K2，触发获取当前时间，因保护故获取失败，触发总线故障，LED_RED亮起；
8）再次按下K10，切换当前线程到特权模式，LED_BLUE亮起，继续完成完成之前获取时间的任
务，获取成功后通过串口打印输出；
9）可以重复测试步骤5-8。
