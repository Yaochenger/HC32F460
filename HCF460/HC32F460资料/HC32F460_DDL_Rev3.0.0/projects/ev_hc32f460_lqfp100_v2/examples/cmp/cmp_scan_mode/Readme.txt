平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示CMP的扫描比较功能。
说明：
样例连续扫描检测INP1和INP2，通过VCOUT引脚，查看检测状态。

使用步骤
================================================================================
1）打开工程并重新编译;
2）连接PA4(CMP2_INP1)与PC0(ADC电位器输出),
   示波器连接PB13(VCOUT2)查看输出；
   调节ADC电位器为最小值；
   PA5(CMP2_INP2)连接稳压直流电源（<VCC）
3）全速运行样例代码。
4）调节稳压直流电源使CMP2_INP2输入大于VCC/2，VCOUT2输出连续脉冲方波，表示INP2电压输入高于INM电压；
   调节稳压直流电源使CMP2_INP2输入小于VCC/2，VCOUT2输出低电平，表示INP2电压输入低于INM电压
5）调节稳压直流电源使CMP2_INP2输入小于VCC/2，调节ADC电位器为最大值，VCOUT2输出连续脉冲方波，表示INP1电压输入高于INM电压
   调节稳压直流电源使CMP2_INP2输入小于VCC/2，调节ADC电位器为最小值，VCOUT2输出低电平，表示INP1电压输入低于INM电压
6）重复步骤4）~步骤5）
