平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER6的PWM的锯齿波模式、无缓存输出功能，
说明：
配置timer6，timer6计数时钟设置为200M/1024
================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） 使用示波器测试PA8(Timer6_1 PWMA)和PE8(Timer6_1 PWMB)的输出波形，看到PWMA和PWMB输出PWM波形，
    且PWMB的高电平时间为PWMA的低电平时间的两倍。
3） 板子上的蓝色LED以2Hz频率闪烁。
