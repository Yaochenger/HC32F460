平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER62的PWMA端口的捕获输入功能(双缓存功能）
说明：
配置timer6，计数时钟为200M/1024

================================================================================
使用步骤
================================================================================
1）硬件短接PA8(Timer6_1 PWMA)和PA9(Timer6_2 PWMA)
2）打开串口助手软件，配置端口如下参数：
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
3）打开工程编译并全速运行
4）Timer6_1的PWMA引脚交替输出3种占空比的PWM波形（比较值分别为0x2000,0x4000,0x6000，周期为0xFFFF）
5）在串口调试助手可看到u32CaptureA,u32CaptureC,u32CaptureE的值交替变化，
    且值与timer6_1的比较值一致
    且缓存传递方向为u32CaptureA-->u32CaptureC-->u32CaptureE
