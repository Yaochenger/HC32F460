平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER6的PWM的三角波模式、单缓存输出功能和死区功能，
说明：
配置timer6，计数时钟设置为200M/256，
PWMA,PWMB分别配置为比较模式,单缓存输出，并且在下溢中断内改变缓存内比较值。
死区功能配置为DTDAR和DTUAR自动相等，buffer功能不使能。

使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） 使用示波器测试PA8（PWMA）和PE8（PWMB）的输出波形，PWMA和PWMB的反向且，占空比
    每隔两个周期循环一次。
    且因为死区功能的作用，PWMA的上升沿比PWMB晚2.6mS左右，PWMA的下降沿比PWMB早
    2.6mS左右。
