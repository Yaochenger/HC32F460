平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示MCU的TIMER6的硬件启动、停止、清零功能
说明：
配置timer6，timer6计数时钟设置为200M/1024
PA8(PWMA)
PE8(PWMB)

硬件启动: PWMA 上升沿

硬件停止: PWMA 下降沿

硬件清零: PWMB 上升沿

================================================================================
使用步骤
================================================================================
1）硬件连接
   PA8(PWMA)--PE6
   PE8(PWMB)--PD7
2）打开串口助手软件，配置端口如下参数：
   波特率：115200
   数据位：8
   校验位：None
   停止位：1
3）打开工程编译并全速运行。
4）串口调试助手查看定时器工作情况及输出的计数值，
   开始计数后，打印出20次计数值，计数值不相等；
   停止计数后，打印出的5次计数值，计数值相等；
   清除计数值后，打印出5次计数值，均为0。
   如下：
        Start count, CNTER =
        0x155
        0x7be1
        ......
        Stop count, CNTER =
        0xa239
        0xa239
        0xa239
        0xa239
        0xa239
        Clear count, CNTER =
        0x0
        0x0
        0x0
        0x0
        0x0
