平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例为系统时钟切换，并8分频后，通过MCO1(PA8)输出，上电时默认系统时钟为MRC。
通过按键选择系统时钟：
HRC     16     MHz  DIV8    2   MHz
MRC     8      MHz  DIV8    1   MHz
LRC     32.768 KHz  DIV8    4096Hz
XTAL    8      MHz  DIV8    1   MHz
XTAL32  32.768 KHz  DIV8    4096Hz
MPLL    200    MHz  DIV8    25  MHz

================================================================================
使用步骤
================================================================================
1）连接PA8与示波器；
2）打开工程并重新编译下载后全速运行；
3) 通过按键K10切换系统时钟频率；
4）查看示波器,可观察到波形及频率变化。
