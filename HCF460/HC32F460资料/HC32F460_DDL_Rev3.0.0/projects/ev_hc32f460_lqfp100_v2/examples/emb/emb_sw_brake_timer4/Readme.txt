平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示EMB模块的的软件触发制动信号功能。

无

无

================================================================================
使用步骤
================================================================================
1）测试板PA8(TIM4_<t>_OUH)、PA7(TIM4_<t>_OUL)引脚与示波器相连。
2）编译、下载并运行。
3）观察示波器，TIM4_<t>_OUH、TIM4_<t>_OUL输出PWM。
4）按下按键K10，释放。
5）观察示波器，TIM4_<t>_OUH、TIM4_<t>_OUL输出低电平。
6）按下按键K10，释放。
7）观察示波器，TIM4_<t>_OUH、TIM4_<t>_OUL恢复PWM输出。
8）重复步骤4） ~步骤7）。
