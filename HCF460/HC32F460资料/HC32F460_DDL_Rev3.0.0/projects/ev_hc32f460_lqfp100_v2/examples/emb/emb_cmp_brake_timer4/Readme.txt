平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示EMB模块的电压比较器比较结果制动功能。

说明：
EMB检测电压比较器比较结果标志位。
1、通过DAC生产比VCC/2电压，输入给比较器负端，作为参考电压。
2、比较器比较结果标志位置起时，PWM输出指定电平，产生EMB中断，执行EMB中断服务回调
   函数。
3、EMB中断回调函数内，然后清除比较器比较结果标志位和EMB状态，当比较器正端电压低于
   负端电压时，PWM正常输出。

================================================================================
使用步骤
================================================================================
1）测试板PA8(TIM4_<t>_OUH)、PA7(TIM4_<t>_OUL)引脚与示波器相连，PA4(CMP2_INP1)与
   PC0(ADC电位器输出)。
2）编译、下载并运行。
3）逆时针旋转电位器至最小阻值。
4）观察示波器，TIM4_<t>_OUH、TIM4_<t>_OUL输出PWM。
5）顺时针旋转电位器至最大阻值。
6）观察示波器，TIM4_<t>_OUH、TIM4_<t>_OUL输出低电平。
7）重复步骤3）~步骤6）。
