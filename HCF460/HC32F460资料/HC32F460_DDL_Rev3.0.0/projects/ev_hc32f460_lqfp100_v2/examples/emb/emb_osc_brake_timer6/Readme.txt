平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示EMB模块的XTAL故障刹车功能。

说明：
EMB检测外部高速振荡器XTAL故障。
1、外部高速振荡器XTAL故障时，PWM输出指定电平，产生EMB中断，执行EMB中断服务回调函数。
2、EMB中断回调函数内，然后清除XTAL故障标志和EMB状态，PWM正常输出。

================================================================================
使用步骤
================================================================================
1）测试板PA8(TIM6_<t>_PWMA)、PA7(TIM6_<t>_PWMB)引脚与示波器相连。
2）编译、下载并运行。
3）观察示波器，TIM6_<t>_PWMA、TIM6_<t>_PWMB输出PWM。
4）短路外部晶振电容C78的两端（人为制造XTAL故障），然后释放。
5）观察示波器，TIM6_<t>_PWMA、TIM6_<t>_PWMB输出低电平。
6）按下按键K10，释放。
7）观察示波器，TIM6_<t>_PWMA、TIM6_<t>_PWMB恢复PWM输出。
8）重复步骤4） ~步骤7）。
