平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为I2C外设主机模式进行查询方式的数据收发样例

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译。
2）准备另一个测试板下载样例i2c_slave_polling。
3）连线：
   主机        从机
   J3 SDA  ->  J3 SDA
   J3 SCL  ->  J3 SCL
   J3 GND  ->  J3 GND
4）全速运行从机板子的i2c_slave_polling样例。
5）全速运行主机样例,短按按键K2启动数据发送接收。
6）如数据发送接收成功，主机板子蓝灯闪烁；如果失败，红灯闪烁。
