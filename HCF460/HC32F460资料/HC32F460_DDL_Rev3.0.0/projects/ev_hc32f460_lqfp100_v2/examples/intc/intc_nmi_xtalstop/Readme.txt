平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例展示INTC模块NMI_XTALSTOP的检测功能。

说明：
本样例通过人为制造XTALSTOP故障，展示MCU的NMI_XTALSTOP中断功能。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序并运行，发现蓝色LED闪烁；
3）人为制造XTALSTOP故障，如镊子短接外部晶振电容C77的两端；
4）观测LED状态：蓝色LED停止闪烁，红色LED闪烁；
5）撤销XTALSTOP故障，红色LED熄灭，蓝色LED闪烁。

================================================================================
注意
================================================================================
1. 使用NMI XTALSTOP功能时，需要先启动XTAL并等待其振荡稳定，然后使能XTALSTD功能，
并选择发生XTALSTOP故障后产生中断，而非复位；
2. 发生XTALSTOP故障进入NMI中断后，需要清除XTALSTD故障中断标志，才能再次进入NMI中断。
