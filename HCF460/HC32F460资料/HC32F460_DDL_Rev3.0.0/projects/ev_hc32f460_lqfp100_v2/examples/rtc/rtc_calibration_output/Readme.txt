平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示RTC时钟误差补偿及1Hz输出功能。

说明：
本样例设置RTC的时间格式为24小时制，通过按键K1和K2增减补偿值，并设置端口PC13为
1Hz输出端口，同时LED_RED每2秒闪烁一次。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
数字万用表

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）将数字万用表连接到PC13端口，测量频率为1Hz；
4）连续点击按键K1或K2，观察测量的频率的变化；
5）同时观察LED_RED慢速闪烁。
