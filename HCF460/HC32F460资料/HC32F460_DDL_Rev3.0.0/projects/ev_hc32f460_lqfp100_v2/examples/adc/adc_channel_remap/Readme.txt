平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程以ADC单元1（CM_ADC1）为例，将模拟输入引脚ADC1_PIN10（PC0，默认通道10）映射到ADC1的
通道0，即，扫描重新映射通道后的通道0，即是扫描引脚PC0。

说明：
1. 只有通道0至通道15可以重映射；
2. 只有引脚0至引脚15可以重映射；
3. 同一个引脚可以映射至多个通道，但必须注意，不能在多个单元同时对同一个模拟输入引脚进行
   扫描。

================================================================================
使用步骤
================================================================================
1）用 DAP 接口调试，可以直接通过串口调试助手查看程序运行情况；
2）打开工程，重新编译，启动调试或直接下载程序运行；
3）通过串口调试助手查看 ADC 值和对应的电压值；
4）旋转电位器 R76 ADC，可看到变化的 ADC 值和对应的电压值。
