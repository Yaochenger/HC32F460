平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程介绍了PGA的配置方法。例程设置PA0放大倍数为2.286，并用串口将模拟输入信号放大
前和放大后的电压值输出。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
杜邦线

辅助软件:
---------------------
串口调试助手软件（19200bps，8N1）

================================================================================
使用步骤
================================================================================
1）用 DAP 接口调试，可以直接通过串口调试助手查看程序运行情况；
2）将 PA0 与 PC0 短接，以便用电位器调节输入电压；
3）打开工程，重新编译，启动调试或直接下载程序运行；
4）旋转电位器 R76 ADC，调节 PGA 输入电压，用串口调试助手观察模拟输入信号经 PGA 放大
   前后的电压值。

================================================================================
注意
================================================================================
PGA 输入电压范围：最小 (0.1 * AVCC) / Gain；最大 (0.9 * AVCC) / Gain。其中，Gain是
增益系数（放大倍数）。
