平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程展示了用硬件触发源触发ADC扫描的实现方式。

说明：
1. ADC工作在中断模式；
2. 序列A的触发源为ADTRG(PE7)引脚的下降沿，序列B的触发源为Timer0单元1通道B的比较匹配
   事件（该事件每秒产生一次，即序列B每秒扫描一次）。

================================================================================
使用步骤
================================================================================
1）用 USB 数据线连接 USBDBG MicroUSB，可以通过串口调试助手查看程序运行情况；
2）将 PE7 与 PB1（连着按键 K10）短接；
3）打开工程，重新编译，启动调试或直接下载程序运行；
4）串口调试助手每秒输出一次序列 B 的触发情况；
5）按下按键 K10，串口调试助手将输出序列 A 的触发情况。
