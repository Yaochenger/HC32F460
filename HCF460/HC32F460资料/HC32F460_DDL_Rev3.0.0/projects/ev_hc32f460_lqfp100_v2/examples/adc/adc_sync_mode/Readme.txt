平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程使用了全部的ADC单元来演示ADC的协同模式，并将各ADC的扫描时序通过引脚PB5和PD8输出，
可通过示波器观察。

说明：
工作在协同模式的ADC单元：
1. 只使用序列A，禁止序列B；
2. 禁止同一引脚在不同单元同时被采样；
3. 应使用相同的初始化配置；
4. 序列A使能相同的通道数；单元之间，序列A使能的通道可以不相同，但通道间的相对位置应相同。
   如：ADC1使能通道1000010100000000，ADC2可使能通道0000000010000101或0000001000010100等
5. 各通道的采样时间应相同。
6. 例程中当ADC和ADC2协同工作，且协同模式为循环延迟触发模式时(ADC_SYNC_MD == ADC_SYNC_SINGLE_DELAY_TRIG)，
   只使能了一个ADC通道，且其对应的是同一个外部模拟输入引脚，这样配置的目的是为了实现对同
   一个模拟输入引脚以5Mbps的速率进行采样。

================================================================================
使用步骤
================================================================================
1）将示波器探头连接至 PB5（指示 ADC1 采样时序），PD8（指示 ADC2 采样时序）;
2）将 PE7 与 PB1（连着按键 K10）短接；
3）打开工程，重新编译，启动调试或直接下载程序运行；
4）按下按键 K10，触发各 ADC 单元扫描，示波器显示两路频率为 1.25MHz 的 PWM。
