平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程展示了ADC模块的基本用法：用序列A单次扫描模式对指定的ADC通道进行扫描。

说明：
1. ADC工作在轮询模式；
2. 其中一个ADC通道对应的引脚连接了一个电位器，例程将该通道的ADC值和电压值通过串口输出。
   使用时，旋转电位器，可在上位机通过串口调试助手看到变化的ADC值和电压值；
3. 例程还展示了ADC模拟电路时钟ADCLK的配置方式。一般地，在系统时钟不高于60MHz时，可使用
   默认配置，而不需对ADC时钟进行配置。若系统时钟高于60MHz或使用PLL时钟作为ADCLK时，请
   参考用户手册描述的ADC时钟的的条件，进行正确的配置；
4. 宏 ADC_VREF 指定目标板参考电压。

================================================================================
使用步骤
================================================================================
1）用 DAP 接口调试，可以直接通过串口调试助手查看程序运行情况；
2）打开工程，重新编译，启动调试或直接下载程序运行；
3）通过串口调试助手查看 ADC 值和对应的电压值；
4）旋转电位器 R76 ADC，可看到变化的 ADC 值和对应的电压值。
