平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示如何使用TMR4脉冲宽度调制模块PWM死区定时器模式。

说明：
样例配置比较输出模块的低通道，并通过PWM死区定时器模式，输出其波形。

无

================================================================================
使用步骤
================================================================================
1）测试板J12-PE9(TIM4_<t>_OUH)、J12-PE8(TIM4_<t>_OUL)引脚与示波器相连。
2）编译、下载并运行。
3）观察示波器，TIM4_<t>_OUH、TIM4_<t>_OUL通道波形频率为16HZ。TIM4_<t>_OUH占空比
   为1/2，TIM4_<t>_OUL占空比为1/4。
