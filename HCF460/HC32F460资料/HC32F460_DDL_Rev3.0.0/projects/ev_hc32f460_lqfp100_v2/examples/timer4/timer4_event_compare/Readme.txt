平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示如何使用TMR4事件模块比较触发模式。

说明：
TMR4事件模块比较触发模式：计数器单元与事件单元寄存器值匹配时，产生事件和信号，
该事件启动ADC采样。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
串口助手软件

================================================================================
使用步骤
================================================================================
1）通过USB数据线连接PC和板子DAP。
2）打开串口助手软件，配置端口如下参数：
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
3）编译、下载并运行。
4）旋转可调电阻R76。
5）通过串口助手，查看ADC采样值相应跟随改变。
6) 重复步骤4） ~步骤5）。
