平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示I2S半双工输出音频功能。

说明：
本样例配置I2S工作在只发送模式，使用内部时钟，音频采样频率位32K，通过耳机播放声音。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
耳机

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）将目标板的J9拨码开关拨到MCK一端，然后将耳机插入J15（HPHONE）端口；
2）打开工程并重新编译，启动IDE的下载和调试功能，关闭IDE下载界面；
3）戴上耳机后，按下按键K2，在耳机中听到“测试通过”的声音表示功能正常。
