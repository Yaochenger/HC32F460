平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示I2S全双工录音和同步播放功能。

说明：
本样例配置I2S工作在全双工模式，使用外部时钟，音频采样频率位32K，通过DMA实现录音
同步播放功能。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
耳机

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）将目标板的J9拨码开关拨到XTI一端，然后将将耳机插入J15（HPHONE）端口；
2）打开工程并重新编译，启动IDE的下载和调试功能，关闭IDE下载界面；
3）按下按键K2，观察LED_BLUE亮起，在MIC1处说话或者播放音乐，可以在耳机中听到相应的声
音，则功能正常；
4）按键按键K3，观察LED_BLUE熄灭，在耳机中听不到声音，则功能正常；
5）步骤3-4可以重复测试。
