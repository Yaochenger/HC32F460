平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示了AES加解密。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------

辅助软件:
---------------------
串口调试助手软件（19200bps，8N1）

================================================================================
使用步骤
================================================================================
1）用 DAP 接口调试，可以直接通过串口调试助手查看程序运行情况；
2）打开工程，重新编译，启动调试或直接下载程序运行；
3）用串口调试助手或断点方式查看加密和解密运行的结果：
   加密验证成功，打印输出“AES encryption OK.”；否则打印输出“AES encryption FAIL.”，
   程序停止运行；
   解密验证成功，打印输出“AES decryption OK.”；否则打印输出“AES decryption FAIL.”，
   程序停止运行；
