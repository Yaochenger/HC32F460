1. 使用 CDM212364_Setup.exe 安装先楫半导体的调试芯片的驱动
2. 此时已经有驱动，但是该驱动不能不用调试，需要使用 zadig-2.7.exe 进行驱动替换
3. 软件点开后，在选项卡中选中 Options ，并打开 List All Device 选项；操作后方可看到正在使用的USB驱动程序
4. 按照FTDI_device.png 的内容来替换驱动，Attention：一定要注意，否则可能替换为鼠标或者键盘驱动，那样鼠标键盘就没有办法使用了。


5. 如果真的不幸改掉了鼠标/键盘的驱动，那么打开计算机管理->设备管理器->对应被更改驱动的USB的设备->退回驱动版本 可以解决这个问题，
虽然说着轻飘飘，但是定位这个被更改驱动的USB设备需要花费比较久的时间。
